## include機能を利用してflag.txtを読み込む
$_GETで、「🤯」に指定されたものはそのまま読み込まれるコードになっているため、「?🤯=../../../../../flag.txt」のようにディレクトリトラバーサルを利用し、URLを変更すれば、flag.txtを読み込める。
